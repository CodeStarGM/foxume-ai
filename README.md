# 📄 Foxume AI

Analyze your resumes instantly and see how ATS-friendly they are! 🚀

This app lets users **upload their resume**, get a **score**, and see a detailed **resume overview** to optimize for recruiters and ATS systems.

## ⚡ Features

- Upload resume in PDF/DOCX format
- Get an instant ATS score
- View detailed feedback and suggestions
- Optimized for tech and non-tech resumes

## 🚀 How to Use

1. Go to the **Upload Resume** page
2. Select your resume file and click **Analyze**
3. Check your **Resume Overview** page for your score and tips
